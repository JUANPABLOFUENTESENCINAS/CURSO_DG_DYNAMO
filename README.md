# CURSO DE BASICO DE DYNAMO 
![Texto alternativo](DYNAMO.PNG)
## ¡Bienvenido al inicio de la programación visual!

Este repositorio contiene todo el material didáctico, scripts descargables (`.dyn`) y guías paso a paso del curso **Introducción a Dynamo para Revit 2026 (Nivel Básico)**. El objetivo de este programa es transformarte de un usuario tradicional de Revit a un modelador/coordinador eficiente capaz de automatizar tareas repetitivas, gestionar datos masivos y generar geometrías complejas mediante nodos.

---

## 1. Detalles y Requisitos del Sistema
* **Nivel:** Básico / Inicial (No se requieren conocimientos previos de programación).
* **Software Base:** Autodesk Revit 2026.
* **Entorno:** Dynamo Sandbox / Dynamo para Revit v3.x.
* **Idioma:** Ingles.

---

## 2. Contenido del Curso
* **UNIDAD 1:** Nodos y principios basicos con Dynamo
* **UNIDAD 2:** Listas
* **UNIDAD 3:** Curvas
* **UNIDAD 4:** Diseño Parametrico

---
